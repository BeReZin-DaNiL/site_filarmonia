# Информационно-справочная система филармонии

## Описание проекта
Веб-приложение для филармонии, позволяющее пользователям просматривать афишу мероприятий, бронировать и покупать билеты, читать новости, а администраторам — управлять контентом и просматривать отчеты.

## Структура
Проект состоит из двух основных частей:
1. **Frontend**: React приложение (Vite + Tailwind CSS).
2. **Backend (Mock Server)**: Node.js сервер, эмулирующий работу реального бэкенда (API, база данных в памяти).

## Требования
- Node.js (версии 16 или выше)
- npm (обычно устанавливается вместе с Node.js)

## Инструкция по запуску

Для работы приложения необходимо запустить **одновременно** два процесса в разных терминалах: Mock Server (Бэкенд) и Frontend.

### Шаг 1: Запуск Mock Server (Бэкенд)

1. Откройте терминал.
2. Перейдите в папку сервера:
   ```bash
   cd frontend/mock-server
   ```
3. Установите зависимости (только первый раз):
   ```bash
   npm install
   ```
4. Запустите сервер:
   ```bash
   node server.js
   ```
   *Сервер запустится на порту `8080`. Вы увидите сообщение: `Mock API server running on http://localhost:8080`.*

### Шаг 2: Запуск Frontend (Клиентская часть)

1. Откройте **новый** терминал (не закрывая предыдущий).
2. Перейдите в папку фронтенда:
   ```bash
   cd frontend
   ```
3. Установите зависимости (только первый раз):
   ```bash
   npm install
   ```
4. Запустите режим разработки:
   ```bash
   npm run dev
   ```
   *Приложение будет доступно по адресу `http://localhost:5173`.*
